Lab 1

Artemis Setup

I was able to successfully connect to the Artemis Nano module and run the Blink Example.

I ran the Example2_Serial script and confirmed that the serial port was working.

I also ran the analogRead example and could read out the voltage and temperature of the module.

Lab 2

Bluetooth Communication

This lab involved testing the low-latency, moderate-throughput wireless communication between the Artemis board and a computer via Bluetooth LE. The main files for this lab were ECE_4960_Robot.ino, which was the Arduino sketch to be uploaded, and main.py, which was the Python Bluetooth example.

I first set up the USB passthrough to my Ubuntu VM through the VirtualBox Extension Pack and also installed Bleak, a GATT client software capable of connceting to BLE devices. After setting up the USB passthrough, I was able to download the distribution code and run main.py to try to discover my robot, caching the address "66:77:88:23:BB:EF" to Settings["cached"] in settings.py

Put in picture of pong times

I was able to successfully discover my robot and ping it.

On average, the round trip latency is around 0.0236 seconds. As shown in the screenshot, the general pattern for latency times seems to be that there is one short round trip delay that is followed by a significantly longer one. THe ping sends a packet over Bluetooth of 99 bytes and is slower than a wired, serial connection.

I ran the Example2_Serial script and confirmed that the serial port was working.
